# ぽいっとHP サイト仕様（単一ソース・毎回これを参照）

会長決定（2026-07-09）。サービス名＝**ぽいっとHP**（mono.create）。「ぽいっと投げるだけ＝めんどくさいを丸投げ」。
> ⚠️ 過去、能書き構成・複数ページ・制作実績の見せ方を毎回作り直して要件を落としていた。**着手前に必ず本ファイルを読む。**

## ポジショニング（[[project_positioning_local_shops]]）
- 理念：地域で頑張るお店が“本当に必要なこと”に集中できる環境をつくる。
- 訴求主軸：「めんどくさいは、ぜんぶ丸投げでいい」。安さより“手間ゼロ”。ターゲット＝地域の実店舗。
- 2026-07-10改訂: 営業は**当面旭川フォーカス**（全国は第2フェーズ）。**写真対応あり**（スマホ写真OK・なければ撮影相談/旭川近郊訪問可）。

## ページ構成（**複数ページ**・できるくんLPと同じ構成を分割）
参考：`https://lp.dekirukun-dx.com/lp/hp-dekirukun/`
1. **index（トップ）**：ヒーロー(キャッチ＋料金訴求＋CTA) / こんなお悩み / その悩みを解決する3つの強み(丸投げ・高品質・安心サポート) / 制作実績プレビュー(カルーセル) / 無料相談CTA
2. **works（制作実績）**：業界別の端末モックを多数。**カルーセルで自動スライド**
3. **price（料金・他社比較）**：¥49,800(制作)＋年間¥10,000(維持)。他社サービスとの比較表
4. **flow（制作の流れ）**：ヒアリング→制作→納品→更新丸投げ（できるくんの「今後のスケジュール」に相当）
5. **faq（よくある質問）**
6. **contact（無料相談）**：相談・資料請求フォーム＝**受託導線**
- 全ページに「今すぐ相談する」「資料請求」CTAを繰り返し配置（できるくん流）。

## 制作実績の見せ方（必須）
- **PC＋スマホの端末モック＋業種ラベルタブ**（できるくん同様）に実サイトを表示。
- **モーション＝hidamari-home.jp のEvent欄と同じ**：Swiperループ自動スライド（3秒ごとに2秒かけて送る・中央寄せ）＋スクロールfade-up。制作実績カードをこれで動かす。
- 本番は**各サイトのスクショを焼き込み**（ライブiframe多数は重く不確実。軽く確実に全枚表示）。

## 制作ルール（厳守）
- 素材イラスト＝**Tech Pic**（商用可・加工可・**1サイト20点まで**・800px・クレジット不要）。
- 執筆＝**反AI検出ルール(§3-1)**・**絵文字禁止**・`jp_lint`違反0。
- 実在店の“営業用サイト”を作る場合は**その店の実写真のみ**（他店/汎用/生成画像の流用は不可）。
- ポートフォリオ(架空店)の代表画像も**実写のフリー素材のみ**（Openverse CC0 / Wikimedia Commons / Pexels 等・商用可）。**AI生成に見える食品・人物・店内写真は使用禁止**（ツヤ過剰・不自然な湯気/麺・完璧すぎる構図はAIサインとして弾く）。採用前に必ず目視でAIっぽさを確認する。会長が「AIっぽい」と感じた時点で不合格。
- 品質基準＝`品質基準_ポートフォリオ.md`（ramen水準）。
